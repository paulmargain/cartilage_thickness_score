        <h2 style="margin-top:0;">TL;DR</h2>

        <p>

          We propose an automatic, scalable framework that builds high-resolution cartilage thickness maps (CTh-Maps) from knee MRI

          and predicts a continuous OA cartilage severity score (CTh-Score, 0–100).

        </p>


        <h2>Abstract</h2>

        <p>

          To develop and validate an automatic, scalable framework for assessing the femoro-tibial osteoarthritic cartilage severity

          using high-resolution cartilage thickness maps (CTh-Maps) and a cartilage thickness scoring system (CTh-Score).

        </p>


        <h2>Method (overview)</h2>

        <ul>

          <li>3D-UNet segments femoro-tibial bones + cartilages across OAI follow-ups (baseline, 1, 2, 3, 4, 6, 8 years).</li>

          <li>Generate CTh-Maps for each knee.</li>

          <li>ResNet predicts a CTh-Score from 0 (healthy) to 100 (end-stage OA).</li>

          <li>Validate reproducibility + correlation to expert cartilage loss (MOAKS) and OA severity (KL grade), incl. external dataset.</li>

        </ul>


        <h2>Key results</h2>

        <ul>

          <li>OAI-scale dataset generated (“OAI CTh-Maps”).</li>

          <li>Excellent reproducibility for CTh-Maps and CTh-Score (ICC &gt; 0.98).</li>

          <li>Strong correlation with expert cartilage loss (r = 0.81) and association with OA severity (incl. external dataset).</li>

          <li>CTh-Score increases significantly over time and shows sensitivity up to ~6 years prior to KL progression.</li>

        </ul>
